Location Master App Description

Location Master App provides extensive functionality for geo-features, including Points, Paths/Lines, and Polygons. A brief overview of each is given below:
Point:
The application provides real-time details about the current location, including latitude, longitude, altitude, accuracy, and address. Additionally, it allows searching for any other location or place, with all these details calculated automatically. Then, points can be saved along with attribute data.
Latitude and longitude values are supported in multiple units, including decimals, degrees-minutes-seconds, radians, and gradians. Saved points can be displayed on Google Maps, shared, copied, edited, and exported in KML, KMZ, and JPG formats.
Path:
This app enables the digitization of lines/paths directly on the map. Paths can be saved along with relevant attribute data, such as length, title, description, date, and time. Length is automatically calculated and displayed in various units, including inches, feet, yards, meters, furlongs, kilometers, and miles.
Paths are easily modifiable by selecting vertices to delete or reposition. Any adjustments recalculate the length in real-time. There are labels along each side of the path which is showing its length. Toggling option allows the user to turn ON or OFF these side-length-labels.
Paths/routes can also be drawn in real-time using the path tracking feature, which automatically maps the route as traveled. Options to pause and resume tracking ensure flexibility, and tracking continues even when the screen is off or the app is closed.
Saved paths are viewable on Google Maps, and can be edited and shared in formats like KML, KMZ, and JPG.
Polygon:
This app supports digitizing polygons on the map. Polygon can be saved with associated attributes such as area, title, description, date, and time. The area is automatically calculated and can be displayed in units like square feet (ft²), square meters (m²), square kilometers (km²), Marla, and Kanal.
Polygons are customizable by selecting vertices to delete or reposition. Adjustments trigger real-time recalculations of the polygon area. Each side having label showing its length. The side length labels can be toggled.
Polygons can also be drawn in real-time using the polygon tracking feature, which automatically maps the shape as traveled. Pause and resume options are available, and tracking continues even when the screen is off or the app is closed.
Saved polygons can be viewed on Google Maps, edited, and exported in KML, KMZ, and JPG formats.
Other Interesting Features:
1. When saving or updating a point, path, or polygon, the user does not need to manually type the title or description/address. Just speak and speak-to-text feature will convert it to text automatically.
2. Another notable feature is the ability to take pictures, where the user's location details—such as latitude, longitude, altitude, accuracy, address, date, and time—are overlaid on the image.
3. Additionally, users can search for a specific point using latitude and longitude. Other related data, like altitude and address, can be calculated and saved for future reference.
4. The app also provides a smart solution for using its features, especially Google Maps, in scenarios where there is no internet connectivity.
Note: While installing the app, make sure to grant all the necessary permissions requested in the prompts, including location, media, gallery, and camera permissions. The app will create a folder named LocationMaster in the Documents directory, where all the exported KML and KMZ files will be stored. Additionally, another folder with the same name will be created in the DCIM directory to store all the exported images as well as photos taken with the camera in JPG or PNG format.
